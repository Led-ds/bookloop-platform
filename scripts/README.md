# scripts — Utilitários

**Finalidade:** automatizar tarefas repetitivas da plataforma.

**Quando usar:** ao iniciar um projeto ou preparar infraestrutura.

**Como evoluir:** manter scripts pequenos, POSIX quando possível, idempotentes e com `--help`.

| Script | Faz |
|--------|-----|
| `new-project.sh` | Scaffolding de um novo serviço a partir de `templates/`. |
