# Template — GitHub Actions (callers)

Workflows mínimos que **chamam** os reusable workflows da plataforma. Copie para
`.github/workflows/ci.yml` do seu app e ajuste os inputs.

Secrets necessários no repositório do app: `DOCKERHUB_USERNAME`, `DOCKERHUB_TOKEN`.

> Ajuste `<org>` e a tag `@v1` para a versão da plataforma que você consome.
