# github/

**Finalidade:** definições dos **reusable workflows** (CI/CD) consumidos pelos repositórios de app.

> ⚠️ Em um repositório GitHub real, reusable workflows **precisam** ficar em `.github/workflows/`.
> Neste scaffold usamos `github/workflows/` conforme a estrutura pedida — ao publicar, renomeie
> a pasta para `.github/workflows/`. Ver ADR-0004.

Ver [`workflows/README.md`](workflows/README.md).
