# Diagramas

**Finalidade:** representações visuais versionáveis da arquitetura.

**Quando usar:** para comunicar topologia, fluxos e limites de contexto.

**Como evoluir:** prefira **Mermaid** ou **C4 em texto** (versionável em diff) a imagens
binárias. Imagens exportadas, se necessárias, ficam ao lado do fonte.

- [`c4-container.md`](c4-container.md) — C4 nível container.
- [`cicd-flow.md`](cicd-flow.md) — fluxo de CI/CD.
